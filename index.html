DAU vs DUU — Full Time Series

Daily Active Users (DAU) vs Daily Unique Users (DUU)  r = 0.98

DAU counts users who took an active action; DUU counts any unique visitor including passive/background sessions. The near-perfect correlation (r = 0.98) means either metric is a reliable trend proxy — but they are conceptually different engagement signals. DUU consistently runs ~13–14× higher than DAU.

DAU / DUU Ratio — Weekday vs Weekend Pattern

The ratio drops reliably from ~7.3% on weekdays to ~4.0% on weekends and holidays. DUU remains elevated on weekends (background sync, mobile, consumer use) while DAU collapses — confirming DAU is the sharper enterprise-work signal. Holiday dips (Christmas, New Year, July 4th) are clearly visible as multi-day troughs near 3–4%.
File Count as a Storage Predictor — Per-Farm Analysis (636 farms)

File Count vs User Storage (per farm)  r = 0.93

File count explains 87% of user storage variance across farms. This is the strongest single-variable predictor for storage capacity planning — stronger than DAU, region, or segment alone.

File Count vs DB/SQL Storage (per farm)  r = 0.87

DB storage correlation is weaker (r = 0.87) because SQL metadata accumulates overhead from versioning, sharing records, permissions, and tenant config that isn't purely file-driven. User storage is a cleaner signal — it's essentially a direct byte-sum of file content.
